Proyecto piedra, papel y tijera:

Tener en cuenta: {
	vista usuario git hub,
	vista de empate,
	eleccion bot 3 seg mostrar en pantalla;
}

Logica: 
	Cuando el usuario elija una opcion sucedera:
	
		se cambiara la vista mostrando el resultado seleccionado,
		
		se tomara un valor de string o number de un arreglo de 3 ( [1:'papel', 2:'roca', 3:'tijera'] ),
		
		comenzara una cuenta regresiva (4 seg aprox),
		
		se mostrara el resultado del bot,
		
		se evaluara el ganador y perdedor,
		
		se reduce o aumenta el score dependiendo del resultado,
		
		se guardara el resultado del score de manera local,
		
		se mostrara un boton de volver a jugar
